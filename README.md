# Sujan Maji | E-Commerce & Performance Engineer Portfolio

Welcome to my professional software engineering portfolio. This website is a high-performance, single-page application showcasing custom theme case studies, professional timeline journey details, and interactive local simulations for performance audits and visual regressions.

## ⚡ Core Features

- **Vibrant Modern Design**: Sleek dark obsidian and Shopify emerald aesthetics utilizing advanced CSS Grid, custom HSL gradients, glassmorphism, and responsive layouts.
- **Dynamic Particles Canvas**: Interactive background particle system reacting to mouse hover, mouse movement, and system theme settings.
- **Interactive Multi-Theme Controller**: Hot-swappable color palettes (Obsidian Dark, Pure Light, Shopify Emerald) with local persistence.
- **Visual Simulators**:
  - **Performance Audit Sim**: Analyzes templates and generates refactored high-fetchpriority Liquid code replacements.
  - **CI/CD Visual Regression Sim**: Interactive slider allowing side-by-side layout regression checking (mock Playwright build output logs).
- **Premium Branded SVGs**: Built-in, high-fidelity inline SVG logos (Shopify, React with slow-spin micro-animation, AWS, Liquid) and branded social links (GitHub, LinkedIn) with customizable hover state colors.
- **Brand Skill Competencies**: Visual tags categorized by Languages, Frontend, Backend, E-Commerce, and DevOps featuring branded colors and hover glows.

## 🛠️ Technological Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Variables, Flexbox, Grid), JavaScript (ES6+), UMD Lucide Icons
- **Backend**: Node.js, Express.js (Static file routing)
- **Dev**: Nodemon

## 🚀 Local Deployment Setup

To clone and run the portfolio locally, ensure you have Node.js and npm/pnpm installed:

1. **Install Dependencies**:
   ```bash
   pnpm install
   # or
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   *The nodemon process will launch and serve the app at [http://localhost:8082](http://localhost:8082).*

3. **Production Startup**:
   ```bash
   npm start
   ```
